============
Introduction
============

This text file contains the updated list of multiplayer games listed on
the Gamespy websites and moreover their PIDs (MASTERID).
The PID (I think means product ID but it's called also simply ID) is
useful in a pair of things:

- the stats visible on http://archive.gamespy.com/stats/ where you can
  see the servers statistics of each multiplayer game like number of
  servers online and players on them.
  Example: http://archive.gamespy.com/stats/mods.asp?id=1111

- cd-key authorization, as explained in
  http://aluigi.org/papers/gskey-auth.txt the PID is used to
  identify the games during the online verification of the cd-keys of
  the games that use the Gamespy cd-key SDK:
    http://aluigi.org/papers/gshlist.txt

- others that I don't know

This list will be updated periodically and I hope it is useful to
someone: usually statistics are needed by some webmasters while the
cd-key verification is important (or needed?) to the gamers of the
games that use the Gamespy cd-key SDK not only to understand better the
code hidden in them but also for interesting things like the manual
verification of the own cd-key (needed when there are unknown problems
and we want to see the verbose error message) and more.

Anyway check my stuff for more tools and informations about these
things and for updates of this document:

    http://aluigi.org/papers.htm#distrust

Note: possible grammatical errors in the names of the games are not my
      fault but just how they are on Gamespy.
Note: half of these games and pids have been provided by an anonymous
      source
